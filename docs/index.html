Bob-a-gotchi

A pixel-art build buddy that lives in your editor. He codes along while you type, cheers when your AI ships a big block, gets hungry, tells dad jokes, and naps when you idle.

IBM Bob VS Code Cursor VSCodium

Quick Start

Install Bob into your editor with one command — no Marketplace, no account.

curl -fsSL https://raw.githubusercontent.com/comacoded/bob-a-gotchi/main/install.sh | bash

Then reload your editor window and click the hard-hat icon in the activity bar to meet Bob.

Prefer to install by hand?

Grab the .vsix from the latest release, then run
code --install-extension ibm-bob-tamagotchi.vsix --force

What Bob does

Builds while you code

Bob lays bricks as you type. When a big block lands, confetti + "wow great block!"

Gets hungry

He rubs his tummy and a burger button appears. Feed him and he munches: "mmm tasty."

Naps when you idle

Step away and Bob tucks into bed under the moon, breathing softly with a little zzz.

Tells dad jokes

While idling he drops programming dad jokes. "I named a bug 'Steve'. We've been together for weeks."
