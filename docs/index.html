Quick start

Install in one command.

No Marketplace account needed. Run the installer, reload your editor window, and click the Bob icon in the activity bar.

Use the hosted installer to fetch the latest packaged extension.

$curl -fsSL https://raw.githubusercontent.com/comacoded/bob-a-gotchi/main/install.sh | bash

Bob installs into the editors it detects: IBM Bob, VS Code, Cursor, and VSCodium.

Prefer to install by hand?

Grab the .vsix from the latest release, then run code --install-extension ibm-bob-tamagotchi.vsix --force.

The companion loop

Small reactions, real personality.

Bob turns the invisible rhythm of coding into a tiny ambient toy: subtle enough for daily work, expressive enough to make the sidebar worth checking.

Builds while you code

Bob lays bricks as edits land, then settles back down when the keyboard goes quiet.

Spots big AI blocks

Large inserted chunks become a celebration moment instead of silent background work.

Needs occasional care

A simple food loop gives Bob stakes without turning your editor into a chore list.

Rests when you do

Idle time becomes a nap state, with energy recovery waiting for your next session.
